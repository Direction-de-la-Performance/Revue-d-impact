Import des remontées existantesOutil interne : Direction de la Performance
Usage. Importe les remontées déjà saisies (bilan, ambitions, ressources) extraites du classeur Excel, pour donner un point de départ réel aux agences plutôt que de repartir de zéro. Par défaut, une remontée déjà présente en base (déjà modifiée en ligne) n'est pas écrasée.
